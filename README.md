# netscreen

Use a secondary laptop as a second screen for your primary PC/laptop (like deskscreen but using ffmpeg+ssh+screen+mpv)

## Requirements

### Primary PC/Laptop

1. Windows 10
2. Powershell Core
3. Amyuni usbmmidd_v2
4. OpenSSH Client
5. FFmpeg

### Secondary Laptop

1. Linux
2. OpenSSH Server
3. Screen
4. MPV
